# 🗳️ Online Voting System

A web-based Online Voting System designed to simplify and digitize the voting process. This project allows users to register, log in, and cast votes securely while maintaining a structured and user-friendly interface.

---

## 📌 Features

* 🔐 User Authentication (Login & Registration)
* 🧾 Personal Information Collection
* 🗳️ Voting Interface
* 📊 Simple UI for Voting Flow
* 🖼️ Image & SVG Asset Support
* ⚙️ Backend Server Integration (Basic)

---

## 🛠️ Tech Stack

### Frontend

* HTML
* CSS
* JavaScript

### Backend

* Node.js (inside `Server/` folder)

### Other

* Static assets (images, SVGs)

## 🚀 Getting Started

### 1. Clone the Repository

```bash
git clone https://github.com/ShivrajCodes/Online-Voting-System.git
cd Online-Voting-System
```

---

### 2. Run Frontend

Simply open `index.html` in your browser:

```bash
open index.html
```

Or double-click the file.

---

### 3. Run Backend Server

Navigate to the server folder:

```bash
cd Server
npm install
npm start
```

> Make sure Node.js is installed on your system.

---

## 🧠 How It Works

1. User registers via `register.html`
2. Logs in using `login.html`
3. Enters personal details
4. Accesses voting page
5. Casts vote through UI

---

## ⚠️ Limitations

* ❌ No advanced authentication (JWT/OAuth)
* ❌ No database integration (or limited)
* ❌ No vote encryption or blockchain
* ❌ Basic UI/UX
